# Gmail to PDF

Exports emails from your Gmail account to PDF

## Requirements

[Gmail gem](https://github.com/nu7hatch/gmail)
[Prawn gem](https://github.com/prawnpdf/prawn)
[Highline gem](https://github.com/JEG2/highline)
